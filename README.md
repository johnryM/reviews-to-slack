# Reviews to Slack

## How to
1. Create a `data.json` file in the folder root and create object containing the following fields:
~~~~
{
    "webhookUrl" : "https://hooks.slack.com/services/....",
    "testChannel" : "#bleh",
    "liveChannel" : "#blah"
}
~~~~
2. Add your Slack webhook and the corresponding channel names.
